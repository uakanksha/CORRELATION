# GGplot
 GG plot is powerfull tool for the data visualisation in R. 
 GG stands for Grammer of Graphics.
 
 It has three elements. 
 1. DATA (as arguemnet)
 2. GEOMETRY (Eg - GEOM_POINT, GEOM_LINE etc.)
 3. AESTHETIC (aes)
 
 OTHER ASPECTS OF GG_PLOT
 1. Statistics
 2. Co-ordinates
 3. Facets
 4. Themes
